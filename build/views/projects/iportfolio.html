iPortfolio 회사 브랜드 사이트

[기여도] 디자인 (100%), 웹 개발 (100%)
아이포트폴리오는 ‘교육을 개혁한다’는 사명으로 교육 시장에서 영향력을 키우고 있는 Ed-Tech 기업입니다.
입사 전에 개발되었던 브랜드 사이트는 Wix에서 제공하는 템플릿으로 사이트가 개발되었었습니다. 이 과정에서 모바일 디바이스가 지원하지 않는 이슈, 제한된 커스터마이징으로 인한 디자인의 한계로, 변경되었으면 좋겠다는 많은 피드백을 받았습니다.
또 회사의 얼굴인 브랜드 사이트가 오래전에 개발된 상태로 외부에 공개되는 게 줄곧 신경이 쓰여 작업을 시작하게 되었습니다.
서비스 아이덴티티와 화면에 대한 디자인 리뉴얼, 개발을 함께 진행했습니다.

[목표]
홈페이지에 들어오는 대상은 협력/제휴업체, 입사 지원자, 회사에 대한 호기심을 가진 자, 어떤 회사에 다니고 있는지 궁금해하는 임직원의 친지/가족이라 판단했고 회사의 미션과 비전, 새로이 추가된 서비스와 기능, 비즈니스 모델을 기반으로 웹 사이트 리뉴얼을 했습니다.

THE PROJECT

반응형 웹으로 제작해 모바일 디바이스에서 동일한 환경으로 콘텐츠를 볼 수 있게 적용했습니다.
콘텐츠에 관한 정보를 시각적으로 재미있게 표현할 수 있도록 Parallax 스크롤 효과를 넣었습니다.
또 뉴스/채용 게시판 등록 등, 빈번한 업데이트 요청에 빠르게 대응할 수 있도록 Admin 사이트도 구축하며 웹사이트 개선 작업을 진행했습니다.

사이트 바로가기

INITIAL WIREFRAME AND FLOW

홀로 기획, 디자인, 개발을 진행하다 보니, 많은 페이지 작업을 진행하는 것 보다는 양질의 콘텐츠를 효과적으로 표현할 방법에 대해 고민했습니다.
회사가 설립된 기간에 비례해 진행되고 있는 프로젝트 수는 많았고, 비슷한 기능을 가진 형태의 프로젝트를 여러 페이지에 보여주는 것보다 한 장의 페이지에 전달하고자 하였습니다.
또 사내 문화(뉴스, 생활)에 대해 알릴 수 있는 기능을 보여주고 싶었습니다. 그래서 리서치를 통해 페레럴즈 스크롤 디자인이 적합하다고 생각하여 적용하였습니다.

CONCEPT DESIGN

사내 디자인의 경우 BX에 대해 고민했거나, 그 중요성을 알고 있다 하더라고 타 프로젝트와 비교하면 우선순위가 낮다고 여겨져 실행될 여유가 없던 상황이었습니다.
정립되지 않은 아이덴티티의 비주얼 작업은 메인 이미지를 시각화하는 과정에서 수많은 고민을 하게 만들었습니다.
아래 이미지는 회사의 철학과 비전, 핵심 가치 및 E-Tech를 나타낼 수 있는 키워드와 서비스 제품 이미지의 연관성을 매칭해 나온 디자인 컨셉 작업입니다.

WEBSITE DESIGN

좌측의 이미지가 개편 전, 우측 이미지가 리뉴얼 된 화면입니다.
아이덴티티를 보여줄 수 있는 메인 이미지와 회사의 미션과 비전, 가치에 대해 보여줬고 페러럴즈 스크롤 효과를 통해 한 개의 이미지가 여러 개의 이미지로 변하는 듯한 착시 효과를 적용했습니다. 뉴스와 채용 게시판을 만들어 업데이트되는 소식을 실시간으로 볼 수 있게 했습니다. 마지막으로 회사의 일상을 보여줄 수 있게 이미지 슬라이드 기능을 적용했습니다.

MULTI DEVICES

기존에 만들어진 사이트 템플릿은 멀티 디바이스 스크린을 지원하지 않았습니다.
개편한 웹 사이트는 모바일, 테블릿, 데스크탑 등의 모든 화면에서 볼 수 있도록 디자인, 개발 되었습니다.

Admin

빈번하게 발생할 수 있는(채용 및 뉴스 게시글 등록) 상황에 대응하기 위해 Admin 사이트를 만들었습니다.
임직원 계정을 생성해 저를 통하지 않고 게시글을 등록, 수정, 삭제할 수 있게 하였습니다. 채용은 꽤 긴 시간 동안 진행되었고, 회사에 관련된 소식이 온라인 뉴스에 나오는 일이 많았습니다.
그렇기에 Admin 페이지의 기능은 채용 담당자에게도 저에게도 매우 유용한 기능으로 사용되었습니다.
