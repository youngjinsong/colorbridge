FAQ

모바일을 PC와 어떻게 연결 하나요?
다음과 같이 모비즌을 통해 모바일 화면을 PC에서 보고 제어할 수 있습니다.

  • 1. 모바일에서 구글 플레이에 접속해 모비즌앱 다운 및 설치

  • 2. 모바일에서 모비즌앱 실행 (첫 실행일 경우 연결설정 진행)

  • 3. PC에서 모비즌 프로그램 설치(프로그램 다운로드)

  • 4. PC에서 모비즌 프로그램 실행(이메일, 연결암호 입력)

  • 5. USB, WiFi 연결 방식 중 최적의 방식으로 모바일과 PC 자동 연결

PC와 모바일을 USB 케이블을 통해 연결하려면 어떻게 해야 하나요?
USB 케이블 연결을 위해서는 다음과 같이 진행을 하면 됩니다.

1. PC에 해당 모바일 USB 드라비버 설치 (설치 파일은 각 제조사 홈페이지 참조)

  • 2. 모바일의 ‘설정 > 응용프로그램 > 개발 > USB 디버깅’ 체크

  • 3. PC와 모바일을 USB 케이블로 연결

  • 4. PC의 모비즌 프로그램 실행

모비즌 키보드는 어떻게 사용하나요?
‘모비즌으로 PC와 모바일을 연결하시면 기본적으로 영문을 직접 입력하실 수 있지만 한글 입력시에는 다음과 같이 키보드 세팅을 해주셔야 합니다.

  • 1. 한/영 키 누름 (한글 입력상태로 바꿈)

  • 2. 한글을 입력하면 바로 모비즌 키보드 설정 창이 팝업

  • 3. 팝업창에서 ‘키보드 설정에서 Mobizen 체크’ 버튼 클릭

  • 4. 모바일 키보드 및 입력방식 세팅 화면에서 ‘Mobizen’ 체크

    5. 모바일 백 버튼 클릭

  • 6. 팝업창에서 ‘입력 방법에서 Mobizen 체크’ 버튼 클릭

  • 7. 모바일 입력방식 선택창에서 ‘Mobizen’ 선택

  • 8. 한글 입력

    * 한글을 입력하시면 뷰어 상단에 한글이 입력되고, 엔터키를 누르면 모바일에 한글이 표시됩니다.

    * 모비즌 연결이 종료되면 이전에 설정한 입력 방식으로 자동 변경됩니다.

이메일은 어떻게 수정하나요?
이메일은 모바일에 설정된 계정을 기본으로 사용하게 되어 있어 수정이 불가하지만,
만약 모바일에 설정된 계정이 2개 이상일 경우에는 다음과 같은 방법으로 다른 이메일을 사용하도록 수정할 수 있습니다.

  • 1. 모바일의 모비즌앱 실행

  • 2. ‘설정 > 설정 초기화’ 메뉴 터치

  • 3. 설정 초기화 화면에서 ‘초기화’ 버튼 터치 (초기화 완료 후 연결정보 설정 화면으로 이동됨)

  • 4. 연결정보 설정 화면에서 등록되어있는 다른 이메일계정 선택

  • 5. 연결암호 등록

    6.완료

연결암호는 어떻게 수정하나요?
연결암호는 다음과 같은 순서로 원할때 수정할 수 있습니다.

  • 1. 모바일에서 모비즌앱 실행

  • 2. 우측상단 세부 메뉴에서 “연결 암호 변경 » 메뉴 터치

  • 3. 연결암호 4자리 입력

  • 4. 연결암호 4자리 재입력

    5. 완료

모바일에 앱을 실행시켜야 PC와 연결할 수 있나요?
모바일에서 모비즌앱을 설치하시면 기본적으로 앱을 별도로 실행 하지 않아도 PC에서 모바일로 연결이 가능합니다.
모바일과 PC연결 방식은 어떻게 변경하나요?
모바일의 모비즌앱에서 연결방식을 다음과 같이 수정할 수 있습니다.

  • 1. 모바일에서 모비즌앱 실행

  • 2. ‘설정 > PC연결방법’ 터치

  • 3. 원하는 연결방식만 선택

  • 4. ‘저장’ 버튼 터치

특정 PC만 내 모바일에 연결되게 하려면 어떻게 해야 하나요?
보안을 위해 특정 PC만 내 모바일에 접근하게 하기 위해서는 다음과 같은 절차로 모바일에서 접근가능 PC를 등록하면 됩니다.

  • 1. 모비즌을 통하여 모바일과 PC연결

  • 2. 우측상단 세부 메뉴에서 ‘PC등록’ 메뉴 터치


  • 3. 설정 > 등록PC 에서 '등록된 PC만 사용' 옵션 선택

모바일 스킨까지 캡처나 녹화를 하려면 어떻게 해야 하나요?
PC에서 모바일 스킨까지 캡처나 녹화를 하려면 다음과 같이 옵션을 설정하면 됩니다.

  • 1. 설정 > 도구 > 캡처 옵션 또는 녹화옵션 선택

  • 2. '스킨포함' 옵션 선택

    3. 캡처 또는 녹화 실행

PC를 통해 모바일에 들어오는 알림 메시지를 확인하려면 어떻게 해야 하나요?
PC에서 모바일에 들어오는 알림 메시지 (전화, 문자, 이메일 수신, SNS (카톡, 페이스북, 트위터 등), 기타)를 확인하려면 다음과 같이 설정을 진행하면 됩니다.

  • 1. ‘설정 > 일반설정 > 알림사용’ 옵션을 ‘사용’으로 체크

  • 2. 알림사용 설정 창에서 ‘설정하기’ 선택

  • 3. ‘접근성’과 ‘Mobizen’ 옵션 체크

모바일을 사용하지 않을때 PC에 보이는 모바일 화면을 자동으로 잠그려면 어떻게 해야 하나요?
다음과 같이 모비즌을 사용하지 않을때 자동으로 PC뷰어를 잠글 수 있습니다.

  • 1. ‘설정 > 일반설정 > 뷰어 잠금’ 옵션 선택

  • 2. 자동 잠금 시간 선택 (1분, 2분, 3분, 5분, 10분)

모바일 화면 녹화시 마우스 움직임도 녹화하려면 어떻게 해야 하나요?
‘설정 > 도구설정 > 마우스 커서’ 옵션을 ‘보임’으로 체크하면 모바일 화면 녹화시 마우스 움직임도 같이 녹화할 수 있습니다.
