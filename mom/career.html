주식회사 앤은 모바일 서비스 [맘톡]을 개발/운영하고 있습니다.

맘톡은 엄마아빠들의 좀 더 '편리하고 스마트한' 육아를 돕기 위한 큐레이션 서비스를 만들어 나가는 중입니다.

개발자

좀 더 편리하고, 좀 더 스마트한 서비스를 구축해 나가기 위해, 실력있고 욕심있는 개발자분들을 모십니다.

  • web frontend developer (3~5년)
    상세업무
    • 현 맘톡 서비스 WEB 구축 (현재 APP으로 서비스 중)
    • 신규 서비스 테스트/구축을 위한 웹 마이크로 사이트 구축
    • 쇼핑 솔루션의 적용
  • backend developer (3~5년)
    상세업무
    • 개인화 추천 솔루션 개발
    • 맘톡 서비스 백엔드 개선 / 신규 개발
  • 필요기술 (공통)
    • html5 / css3 / jquery / ajax / php
    • Java / Ruby
    • My-SQL, MongoDB
    • Nginx, Tomcat, Ruby on rails 가능자
    우대조건
    • 데이터 분석, 추천 시스템 개발 경험
    • 개인화 관련 기술에 대한 이해 및 구현 경험
    인재상
    • 내가 개발한 프로덕트의 성장을 보고 싶으신 분
    • 빠른 변화와 성장에 함께 하고 싶으신 분
    • Free rider가 되기 보다는, 나의 성과에 대한 인정과 보상에 더 많은 관심이 있으신 분
    채용 형태
    • 3개월 간, 저희 회사와 상호 핏을 맞춰보는 기간을 가진 후, 상호 논의하에 정규직으로 전환될 예정입니다. 이 기간에는, 회사와 개발자님이 서로 열정을 느끼는지 알아보는 것 외에는 월급, 4대보험 적용, 회사 복지 등 모든 조건이 정규직과 동일합니다.
    지원 방법
    • 포트폴리오 (본인이 수행한 프로젝트 및 다인 개발의 경우 본인의 역할 명시) 및 경력을 포함한 이력서를 jobs@mom-talk.com 로 메일 접수
    연봉
    • 경력과 스킬 등에 따라 면접 후 결정 목표 달성 시, 인센티브 지급.

※ 본 채용은 적합한 담당자님을 만날 시 마감됩니다.

앱 디자이너

    • main: APP (WEB) UX 개선 및 신규디자인
    • sub: 운영 페이지 제작 업무
    • 현 디자인 인력: 디자이너 1명, 컨텐츠 디자인 채용중
  • 필요기술 (공통)
    • APP 디자인 경력
    • 편리한 UX 구현에 대한 고집
    우대조건

    • 속도와 Quality 간의 우선순위 조절이 탁월하신 분
    채용 형태

편집 디자이너 (인턴, 아르바이트)

저희가 갖고 있는 text 정보들을 좀 더 효과적으로 전달할 수 있도록 작업해주실 컨텐츠 디자이너를 찾고 있습니다.

  • 컨텐츠 디자인 / 인포그래픽 디자인 (경력무관, 학생가능)
    • main: 맘톡이 보유한 임신육아생활 text 컨텐츠를 유저들이 쉽게 인식할 수 있는 인포그래픽한 이미지 파일로 구성
    • sub: 운영디자인 일부 (이벤트페이지, 배너, 캐릭터 작업 등)
  • 필수 조건
    • 그래픽 툴 (포토샵, 일러스트레이터) 숙련
    우대조건
    • 컨텐츠 에디팅, SNS 컨텐츠 제작 등 경험
    • 영상 편집
    인재상
    • 유머, 뷰티, 기타 SNS 등에서 콘텐츠를 만들고, 편집하고, 드러내는 것을 좋아하시는 분
    • 못생긴 페이지를 참지 못하시는 분
    채용 형태
    • 인턴, 기간제 아르바이트, 파트타임, 재택 근무 가능
